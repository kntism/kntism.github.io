# 产品经理能力模型与成长路径

## 项目简介

这是一个关于产品经理职业发展路径的完整指南页面，详细介绍了产品经理从入门到高级的三个成长阶段，包括核心能力要求、实用工具模型、以及推荐书单。

## 页面特色

- 📱 **响应式设计**: 完美适配桌面、平板和手机
- 🎨 **现代化 UI**: 采用卡片式布局，视觉层次清晰
- 📖 **内容丰富**: 涵盖产品经理成长的完整知识体系
- ⚡ **性能优化**: 快速加载，良好的用户体验

## SEO 优化特性

### 1. 元标签优化
- **Title**: 包含核心关键词和年份，利于搜索引擎识别

- **Keywords**: 覆盖产品经理相关的重要搜索词

### 5. 性能优化

### 6. 导航优化

## 内容概览

### 阶段一：执行与落地（产品专员/助理 PM）
- 核心定位：对内把控
- 重点：原型设计、需求管理、项目执行
- 关键技能：逻辑闭环、学习敏锐度、细致规范

### 阶段二：决策与负责（产品经理/高级产品经理）
- 核心定位：对外沟通
- 重点：需求转换、数据分析、用户洞察
- 关键技能：谈判博弈、数据敏感、非职权影响力

### 阶段三：战略与经营（产品总监/产品 VP）
- 核心定位：全局把握
- 重点：战略规划、商业经营、组织建设
- 关键技能：商业敏锐度、前瞻思维、逆境应对

### 核心心法
- 产品经理的平衡艺术
- 六大平衡原则
- 实用决策工具
